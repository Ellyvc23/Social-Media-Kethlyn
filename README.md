# Kethlyn Vaz | Social Media Strategy 📱✨

Este é o repositório do site portfólio profissional da Kethlyn Vaz, especialista em Social Media, Design e Gestão de Tráfego Pago.

O objetivo deste site é apresentar os serviços, o método de trabalho (Método Direção) e os resultados reais alcançados, servindo como uma vitrine profissional para captação de clientes.

## 🚀 Funcionalidades
- **Design Responsivo:** Adaptado perfeitamente para Computadores, Tablets e Celulares.
- **Menu Mobile:** Navegação intuitiva com "menu hambúrguer" para telas menores.
- **Carrossel de Portfólio:** Galeria de vídeos interativa demonstrando resultados reais.
- **Integração Rápida:** Botões flutuantes e links diretos para o WhatsApp e Instagram.
- **Formulário de Contato:** Funcionalidade de envio de e-mails para orçamentos.
- **Animações (AOS):** Elementos que surgem suavemente ao rolar a página (scroll).

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica do site.
- **CSS3:** Estilização, variáveis de cores (Root), Flexbox, CSS Grid e Media Queries para responsividade.
- **JavaScript (Vanilla):** Lógica do carrossel de vídeos e funcionamento do menu mobile.
- **Bibliotecas Externas:** - FontAwesome (Ícones)
  - AOS Library (Animações de scroll)
  - Google Fonts (Poppins e Sacramento)
